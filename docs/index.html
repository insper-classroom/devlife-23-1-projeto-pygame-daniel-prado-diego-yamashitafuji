BOMBERMAN

Descrição do jogo

Caminhe pelo mapa para tentar explodir o seu oponente

Poderes espalhados pelo mapa te dão mais chances de vitória

Poderes especiais

O poder da bomba permite o jogador ter mais bombas para explodir

O poder do fogo permite que a explosão da bomba do jogador tenha maior alcance

O poder do patins permite que o jogador tenha maior velocidade

O poder do coração permite que o jogador recupere sua vida
